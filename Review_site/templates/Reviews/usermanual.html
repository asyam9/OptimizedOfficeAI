    1. 회원가입 / 로그인 / 마이페이지 / 회원탈퇴

    [ 안내 ] 다른 리뷰 보기 등 기본적인 기능들은 회원가입없이 사용이 가능하지만, 리뷰 업로드 기능은 로그인을 하셔야 이용이 가능합니다.

    [ 회원가입 ]
    1. 우측 상단 Sign up 버튼을 클릭합니다.
    2. ID, 비밀번호, 동일한 비밀번호를 입력합니다.
    ❗️ ID는 다른 사용자와 중복되는 ID는 사용할 수 없으며, 영문, 숫자만 허용됩니다.
    3. [ 가입하기 ] 버튼을 눌러주면 회원가입이 완료됩니다.
    ※ 정상적으로 회원가입이 완료된 경우 아래와 같이 입력폼이 변경됩니다.

    [ 로그인 ]
    1. 우측 상단 Log in 버튼을 클릭합니다.
    2. 회원가입시 사용한 ID와 비밀번호를 입력합니다.
    3. 로그인버튼을 눌러주면 로그인이 완료됩니다.
    ※ 정상적으로 로그인이 된 경우 아래와 같이 메뉴바에서
    　 오른쪽 위에 있던 [ Log in ] / [ Sign up ] 버튼이 사라지고 [ 마이페이지 ] 버튼으로 변경됩니다.

    [ 로그인 과정에서 에러발생시 ]
    3번 과정에서 Log in 접속에 에러가 발생하면, ID와 비밀번호를 다시 한 번 확인하고 입력해주십시오.
    ❗️ 동일한 문제가 지속해서 발생한다면, 상단[ About Us ]⇱로 접속하셔서 메일을 주시면 도와드리겠습니다.

    [ 회원 탈퇴 ]

    탈퇴를 원하시는 경우 로그인 > 마이페이지 > 회원탈퇴 버튼을 클릭하시고, 비밀번호를 입력해주세요.
    회원 탈퇴 버튼을 클릭하면 탈퇴가 완료됩니다.
    ※ 정상적으로 회원 탈퇴에 성공한 경우 아래사진 처럼 입력폼이 변경됩니다.

    [ 로그아웃 ]

    2. 나의 리뷰 업로드 / 수정 / 삭제

    [ 안내 ] 다른 리뷰 보기 등 기본적인 기능들은 회원가입없이 사용이 가능하지만, 리뷰 업로드 기능은 로그인을 하셔야 이용이 가능합니다.
    회원가입 및 로그인 매뉴얼은    목차 1번⇱을 참고하십시오.

    [ 리뷰 업로드 하기 ]
    1. 로그인이 되어있는 상태에서 상단 메뉴바의 [ 나의 리뷰 업로드 ] 버튼을 클릭합니다.
    2. 제목(30자 제한), 내용(1,000자 제한)을 기재하고 별점을 선택한 뒤, 리뷰 사진 칸의 < 파일 선택 > 버튼을 클릭합니다.
    3. 업로드 할 사진을 클릭한 뒤 확인 버튼을 눌러줍니다.
    ❗️ 현재 Ol'reviews에서는 .jpg, .png, .jpeg의 사진만 업로드가 가능합니다.
    　기타 확장자의 경우 해당 확장자로 변환 후 업로드를 진행해주세요.
    4. 리뷰 내용을 꼼꼼히 확인한 후 추가 변경사항이 없으시다면 submit 버튼을 눌러주시면, 정상 제출이 완료됩니다.

    ❗️ 입력을 정상적으로 하였는데도 정상 제출이 안되거나, 오류가 발생하는 경우 업로드해주시는 사진 파일명을 영문으로 바꿔주시고 재시도해주세요.
    　만약 동일한 문제가 지속해서 발생한다면 상단[ About Us ]⇱로 접속하셔서 메일을 주시면 도와드리겠습니다.

    [ 리뷰 수정 / 삭제 하기 ]
    1. 로그인이 되어있는 상태에서 상단의 [ 다른 리뷰 보기 ] 버튼을 클릭합니다.
    2. 내가 작성한 리뷰를 찾아 클릭하여 상세보기 페이지를 열어줍니다.
    3. 우측 하단 수정 또는 삭제 버튼을 눌러 원하는 작업을 진행하여 줍니다.
    ※ 수정/삭제 버튼은 내가 작성한 게시글에서만 표시됩니다.

    ❗️ 오류가 발생하는 경우 상단[ About Us ]⇱로 접속하셔서 메일을 주시면 도와드리겠습니다.

    3. 다른 리뷰 보기

    [ 안내 ] 다른 리뷰 보기는 회원가입 없이 이용이 가능합니다.

    [ 다른 리뷰 보기 ]
    1. 상단 메뉴바에서 [ 다른 리뷰 보기 ] 버튼을 클릭해줍니다.
    2. 원하는 리뷰를 찾아 클릭하거나, filter 또는 sorting 기능을 사용하여 원하는 리뷰를 찾아 조회하여 클릭해줍니다.
    * filter, sorting 기능은 여러개를 중첩하여 사용이 가능합니다.

    이름이 왜 Ol'reviews(올리뷰스) 인가요?

    Ol'reviews는 건강한 리뷰문화를 지향하는 서비스로,
    건강을 대표하는 Olive와 유사한 발음을 차용했으며,
    모든 리뷰를 담는다는 의미의 all+reviews를 뜻하고 있습니다.

    브랜드 컬러에 의미가 있나요?

    Ol'reviews는 브랜드컬러로 녹색과 회색을 사용하고 있습니다.
    녹색은 신선한 리뷰와 건강한 올리브를,
    회색은 편향되지 않은 중립성과 객관성을 의미하고있습니다.

    서비스 이용에 필요한 절차는 어떻게 되나요?

    리뷰 조회는 비로그인으로도 가능하며,
    리뷰 업로드 및 커뮤니티(예정)는 간단한 회원가입 후
    서비스를 이용하실 수 있습니다.

    AI분석 기술의 정확도는 어느 정도인가요?

    저희 AI분석 기술은 최고의 정확성을 자랑하며
    지속적으로 업데이트되고 있습니다.
